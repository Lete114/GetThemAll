# Get Them All [资源来自Co2333](https://github.com/Co2333/GetThemAll)

## 描述

这是一个Chrome扩展下载网站上的所有元素，它不仅支持mp3、mp4、mov、gif、png、还支持m3u8和很多图片、视频文件格式。
